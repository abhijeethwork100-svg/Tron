# Prospera Group - React/Next.js Version

An Apple-style redesign of Prospera Group website built with Next.js 14, TypeScript, and Framer Motion.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Features

- Apple-inspired minimalist design
- Smooth scroll animations with Framer Motion
- Responsive layout
- TypeScript for type safety
- Server-side rendering with Next.js

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Framer Motion
- CSS Modules
